# Getting Started

A short guide file, used to test nested-folder rendering and basic formatting.

## Open something

1. Drop a folder onto the window, or click **📂 open folder**.
2. Click any file in the sidebar tree to render it.
3. Use the **filter** box to narrow a large tree.

## Tips

- Star a file with **☆** to keep it in **Favorites**.
- Switch **theme** and **reading font** from the top-bar dropdowns.
- Press **⛶** for distraction-free full-page reading (**Esc** to exit).

> Tip: this whole guide lives two levels deep (`guides/getting-started.md`) — a
> good check that the tree indents and the breadcrumb-style path display work.
